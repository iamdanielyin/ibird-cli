A command line tool for ibird.

## Prerequisites

[Node.js](https://nodejs.org/en/) (>=7.6.x, 8.x preferred), npm version 3+ and [Git](https://git-scm.com/).


## Install

```shell
$ npm install ibird-cli --global
```

## Usage

### Create a project

```shell
$ ibird new
```

### Display help

```shell
$ ibird help
```
